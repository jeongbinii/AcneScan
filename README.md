# AcneScan

여드름 사진을 올리면 종류와 대처방안을 알려주는 AI 피부 관리 서비스.

## 주요 기능

### 여드름 분석
- 사진 업로드 시 여드름 종류 자동 분류
- 약국 연고, 여드름 패치 등 시중 피부 아이템 조합 기반 해결방안 제시

### 구독 기능 (로그인 필요)
- **여드름 슬롯**: 여드름별로 슬롯을 생성해 개별 관리
- **아이템 저장**: 사용자가 보유한 피부 아이템을 등록해두면 AI가 이를 활용해 맞춤 해결방안 제시
- **AI 채팅**: 슬롯마다 전용 대화창 생성 — 경과 사진을 올리면 AI가 전문가처럼 대처방안을 안내
- **슬롯 관리**: 치료 완료 후 슬롯 삭제 또는 영구 보관함 이동

### 커뮤니티 분석
- 사용자 간 치료 패턴 분석을 통한 효과적인 아이템 추천

## 기술 스택

- **Frontend**: SvelteKit, TypeScript, Tailwind CSS
- **Backend**: SvelteKit (Node adapter)
- **Database**: PostgreSQL
- **Infra**: Docker, Docker Compose
