# AcneScan

사진 한 장, 3초면 여드름 분석부터 맞춤 솔루션까지 — AI 기반 피부 관리 서비스.

## 주요 기능

### 여드름 분석
- 사진 업로드 시 여드름 종류(화농성, 구진성, 블랙헤드 등) 자동 분류 및 심각도 판별
- 약국에서 구할 수 있는 연고, 패치, 세안제 등 시중 아이템 조합 기반 맞춤 해결방안 제시
- 업로드 전 사진 보정(밝기, 대비, 채도, 회전, 자르기) 지원

### 마이홈 (로그인 필요)
- **여드름 슬롯**: 분석 결과를 슬롯에 저장하여 치료 경과를 개별 관리 (D+일수 자동 추적)
- **슬롯 이름/날짜 편집**: 슬롯별 이름 변경 및 시작 날짜 수정 가능
- **여드름 종류 태그**: 여드름별 컬러 태그로 한눈에 구분
- **피부 아이템 등록**: 보유 중인 피부 제품을 등록하면 AI가 기존 아이템을 활용한 맞춤 케어 제안
- **전체/개별 선택 삭제**: 아이템과 슬롯 모두 전체 선택, 개별 선택 후 일괄 삭제 가능

### AI 채팅 상담
- 슬롯마다 전용 대화창 생성
- 경과 사진 업로드 시 사진 보정(자르기, 색보정) 후 AI 분석
- 카메라 촬영 또는 갤러리 업로드 선택 가능
- 사용자 보유 아이템을 참고한 맞춤 상담 응답
- 첫 분석 사진과 결과가 대화 기록에 영구 보존

### 구독 플랜
- Free / Pro / Premium 3단계 플랜
- 비로그인 시 구독 버튼 → 로그인 → 결제 페이지 자동 연결
- 결제 페이지 (데모)

### 페이지 구성
- **랜딩 페이지**: 사진 업로드 → 보정 → AI 분석 → 결과 확인 → 슬롯 저장
- **이용 방법** (`/how`): 3단계 이용 가이드 + 구독 안내
- **기능 소개** (`/features`): 주요 기능 상세 설명
- **마이홈** (`/my`): 피부 아이템 관리 + 여드름 슬롯 관리
- **새 슬롯** (`/my/slots/new`): 사진 업로드 → 보정 → 분석 → 대화방 생성
- **채팅** (`/my/slots/[id]`): 슬롯별 AI 상담 대화창
- **구독** (`/subscribe`): 플랜 선택
- **결제** (`/payment`): 결제 정보 입력 (데모)

## 기술 스택

- **Frontend/Backend**: SvelteKit (TypeScript, Svelte 5)
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL 17 + Drizzle ORM
- **Auth**: Cookie 기반 세션 (Node.js crypto.scrypt 해싱)
- **Infra**: Docker, Docker Compose
- **Package Manager**: pnpm

## 개발 환경 실행

```bash
# 의존성 설치
pnpm install

# PostgreSQL 실행
docker compose -f compose.dev.yml up -d

# DB 스키마 반영
pnpm db:push

# 개발 서버 실행
pnpm run dev
```
